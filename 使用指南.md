## 1. 快速开始

### 环境要求
- Windows 10/11 操作系统
- 无需安装 Python（双击 `启动桌宠.vbs` 即可）

### 启动方式

| 方式 | 操作 | 适合场景 |
|------|------|---------|
| 桌面快捷方式 | 双击桌面 `AI桌宠助手` 图标 | 日常使用 |
| 项目文件 | 双击 `启动桌宠.vbs` | 开发调试 |

### 首次使用
1. 启动后在项目目录创建 `.env` 文件，填入 API Key：
   ```
   DEEPSEEK_API_KEY=sk-你的key
   DEEPSEEK_BASE_URL=https://api.deepseek.com
   ```
2. 如果已有 `.env`，直接启动即可。

## 2. 桌宠操作

### 基本交互

| 操作 | 效果 |
|------|------|
| 🖱️ **点击桌宠** | 弹出迷你输入框，输入问题后回车发送 |
| ✋ **拖拽桌宠** | 在桌面上任意移动位置 |
| 🖱️ **右键桌宠** | 打开功能菜单 |
| ⌨️ **输入 quit** | 退出当前对话 |

### 动画状态

| 状态 | 说明 | 触发条件 |
|------|------|---------|
| 🧍 待机 | 呼吸式微动，随机眨眼/走路/跳跃 | 默认状态 |
| 🤔 思考 | 左右摇摆 + 问号 | 等待 AI 回复时 |
| 💬 说话 | 弹跳 | AI 回复到达时 |
| 😴 睡眠 | 缓慢呼吸 + ZZZ | 60 秒无操作自动进入 |

### 对话气泡
- AI 回复时在桌宠上方显示**小气泡**（5 秒后自动消失）
- 气泡**宽度不超过桌宠角色宽度**，简洁克制
- 想查看完整对话→ 右键 → 打开聊天面板

### Mini Mode（隐藏模式）
- **拖到屏幕右边缘** → 桌宠自动滑出屏幕，只留一个小标签
- **鼠标悬停标签** → 桌宠滑回屏幕
- 右键 → 隐藏到边缘 / 展开桌宠（手动切换）

### 系统托盘
- 任务栏右下角有**小猫脸图标** 🐱
- **左键点击** → 显示/隐藏桌宠
- 桌宠隐藏时收到 AI 回复 → **Windows 系统通知**弹窗提示

### 右键菜单功能

| 菜单项 | 功能 |
|--------|------|
| 🧠 查看画像 | AI 记住的关于你的信息（姓名/偏好/技能等） |
| 📋 查看历史 | 对话历史摘要 |
| 💬 打开聊天面板 | 浏览器中打开完整聊天界面 |
| 📄 上传文档 | 选择 PDF/Word/MD/TXT 文件加载到知识库 |
| 🎨 换肤 | 切换橘猫/黑猫/三花/史莱姆像素皮肤 |
| 🖼️ 自定义图片 | 选择本地图片 → 自动像素化 → 设为桌宠形象 |
| 📌 隐藏到边缘 | 手动触发 Mini Mode |
| ⚙️ 设置 | 打开设置面板 |
| ❌ 退出 | 关闭桌宠（对话自动保存） |

## 3. 聊天面板

### 打开方式
1. 右键桌宠 → **💬 打开聊天面板**
2. 或直接访问：[xiaoheng-ai-pet.streamlit.app](https://xiaoheng-ai-pet.streamlit.app/)
3. 或命令行：`uv run streamlit run panels/chat_panel.py`

### 功能说明

| 区域 | 功能 |
|------|------|
| 📋 **左侧边栏** | 对话历史列表、文档上传、快速操作按钮 |
| 💬 **主聊天区** | 对话消息显示（支持 Markdown 格式） |
| ⌨️ **底部输入框** | 输入问题，回车发送 |

### 流式输出
- AI 回复**逐字显示**（打字机效果），无需等待完整回复
- 工具调用过程实时展示（展开/收起）

### 多 Agent 一键调研
- 左侧边栏点击 **🤖 帮我调研 AI Agent 最新动态** 按钮
- 系统自动启动**研究员 + 写手 + 审核员**三人组
- 完成后在聊天区显示完整报告

## 4. 文档问答

### 支持格式
- PDF（.pdf）
- Word（.docx / .doc）
- Markdown（.md）
- 纯文本（.txt）

### 操作方式

**方式一：桌宠右键上传**
1. 右键桌宠 → **📄 上传文档**
2. 选择本地文件
3. 上传成功后，点击桌宠输入问题，如"文档里讲了什么？"

**方式二：聊天面板上传**
1. 打开聊天面板（右键 → 💬 打开聊天面板）
2. 左侧边栏 **📄 文档管理** 区域上传文件
3. 在聊天输入框中提问

**方式三：对话中加载**
- 输入 `/load 文件路径` 加载文档（如 `/load D:/docs/报告.pdf`）

### 多文档支持
- 可以**连续上传多个文档**，不会互相覆盖
- 提问时会**自动跨文档检索**相关内容

## 5. 多 Agent 调研

### 触发方式
1. 在聊天面板左侧栏点击快速操作按钮
2. 或对话中说："帮我调研 XXX，出一份报告"
3. 或对话中说："帮我做一份 XXX 的深度分析"

### 工作流程
```
你的指令
  ↓
🔍 研究员：搜索网络 → 提取关键信息 → 输出研究摘要
  ↓
✍️ 写手：基于研究摘要 → 撰写结构化报告
  ↓
🔎 审核员：事实核查 → 评分 → 修改建议 → 最终版本
  ↓
📄 完整报告（Markdown 格式，含评分）
```

### 自主工作流
- 对话中说"帮我做一个 XXX 的深度调研"
- AI 会**自己规划步骤、换关键词搜索、评估结果、调整策略**
- 全程不需要人工干预

## 6. 设置面板

### 打开方式
右键桌宠 → **⚙️ 设置**

### 配置项

| 分类 | 设置项 | 说明 | 默认值 |
|------|--------|------|--------|
| 🔑 API | API Key | DeepSeek API 密钥 | 从 `.env` 读取 |
| 🔑 API | Base URL | API 服务器地址 | `https://api.deepseek.com` |
| 🔑 API | 模型 | 可选 flash（快速）或 pro（强推理） | flash |
| 🐱 桌宠 | 语音播报 | 是否朗读 AI 回复 | 开 |
| 🐱 桌宠 | 休眠超时 | 多少秒无操作自动睡眠 | 60 秒 |
| 🐱 桌宠 | 桌宠大小 | 像素尺寸 | 128px |
| 📄 RAG | 文档切分大小 | 每个检索段落的字符数 | 500 |
| 📄 RAG | 检索片段数 | 每次搜索返回几个相关段落 | 3 |

## 7. 常见问题

### Q: 桌宠不回复怎么办？
1. 检查 `.env` 文件中的 API Key 是否正确
2. 检查网络是否能访问 `api.deepseek.com`
3. 查看终端日志（用 VS Code 终端启动）

### Q: 对话气泡不显示？
1. 检查桌宠是否被其他窗口遮挡
2. 桌宠在 Mini Mode 时气泡不会显示，改为系统通知

### Q: 文档上传后搜不到内容？
1. 确保文件格式正确（PDF/Word/MD/TXT）
2. 文件内容不能为空
3. 文件路径不能包含中文（Windows 编码问题）

### Q: 如何查看完整对话历史？
- 右键 → 💬 打开聊天面板 → 左侧栏点击历史会话

### Q: 如何更换桌宠形象？
- 右键 → 🎨 换肤（内置 4 种皮肤）
- 右键 → 🖼️ 自定义图片（选择你自己的图片，自动像素化）

### Q: 桌宠会不会占用太多资源？
- 桌宠仅占用约 200MB 内存（主要是 Python 依赖）
- 空闲时 CPU 占用 < 1%
- 日志文件自动轮转，最大 20MB

### Q: 如何彻底卸载？
- 删除项目文件夹即可，没有注册表残留
